# Smart Support Bot: A Rule-Based University Assistance Chatbot

## Project Overview

Smart Support Bot is a rule-based university assistance chatbot developed using Python and the Django framework. The chatbot provides automated responses to common university-related queries such as admissions, courses, fees, placements, hostel facilities, scholarships, contact information, and examinations.

The chatbot uses basic Natural Language Processing (NLP) techniques, including tokenization, stemming, and keyword matching, to understand user queries and generate relevant responses. Chat history is stored in a PostgreSQL database.

---

## Features

* Rule-based chatbot system
* Basic NLP using NLTK
* Tokenization and Stemming
* Keyword-based response generation
* Student query handling
* User-friendly web interface
* PostgreSQL database integration
* Chat history storage

## Installation

1. Clone the repository.
2. Create and activate a Python virtual environment.
3. Install the required Python packages.
4. Configure PostgreSQL database settings.
5. Run database migrations.
6. Start the Django development server.
